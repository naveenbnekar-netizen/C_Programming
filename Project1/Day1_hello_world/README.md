# Day 01 - Hello World Program

## Problem Statement

Write a C program to print "Hello World" on the screen.



## Objective

* Understand the basic structure of a C program.
* Learn how to use the printf() function.
* Learn the purpose of the main() function.


## Algorithm

1. Start the program.
2. Include the stdio.h header file.
3. Define the main() function.
4. Use printf() to display "Hello World".
5. Return 0.
6. End the program.


## Source Code

See "hello_world.c" in this folder.


## Sample Output

Hello World


## Concepts Learned

* #include <stdio.h>
* main() function
* printf() function
* return 0;


## Key Notes

* stdio.h stands for "Standard Input Output Header".
* printf() is used to display output on the screen.
* main() is the entry point of every C program.
* return 0; indicates that the program executed successfully.
