# Day 10 - Electricity Bill Calculator

## Problem Statement

Write a C program to calculate the total electricity bill based on the number of units consumed by a customer.

The program should:

* Accept the number of electricity units consumed from the user.
* Assume the cost of electricity is **₹6.50 per unit**.
* Calculate the total bill amount using arithmetic expressions.
* Display the units consumed, cost per unit, and total bill amount.
* Validate user input and reject negative unit values.

---

## Objective

* Learn user input and output operations.

---

## Concepts Used

* Variables
* Input/Output Functions (`printf()`, `scanf()`)
* Arithmetic Operators
* Input Validation

---

## Formula Used

Total Bill = Units Consumed × Cost Per Unit

Where:

Cost Per Unit = ₹6.50

---

## Sample Output

### Example

----- ELECTRICITY BILL -----

Cost Per Unit  : Rs 6.50

Total Bill     : Rs 975.00
