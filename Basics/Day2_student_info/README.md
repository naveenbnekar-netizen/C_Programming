# Day 02 - Student Information System

## Problem Statement

Write a C program to create a Student Information System that accepts and displays the following student details:

* Name
* USN
* Branch
* Age
* CGPA

The program should display the entered information in a neatly formatted manner.

## Objective

* Learn how to take multiple inputs from the user.
* Understand the use of different data types in C.
* Practice storing and displaying information using variables.
* Improve output formatting skills using "printf()".

## Algorithm

1. Start the program.
2. Include the "stdio.h" header file.
3. Define the "main()" function.
4. Declare variables for Name, USN, Branch, Age, and CGPA.
5. Prompt the user to enter the student details.
6. Read the inputs using appropriate input functions.
7. Display all the entered details in a formatted output.
8. Return "0".
9. End the program.

## Source Code

See "student_info.c" in this folder.

## Sample Input

```text
Enter Name: Naveen
Enter USN: 4XX22CS001
Enter Branch: ECE
Enter Age: 23
Enter CGPA: 8.75
```

## Sample Output

```text
------------------------------
STUDENT DETAILS
------------------------------
Name   : Naveen
USN    : 4XX22CS001
Branch : ECE
Age    : 23
CGPA   : 8.75
------------------------------

## Concepts Learned

* Variables and Data Types
* User Input using `scanf()`
* Output Formatting using `printf()`
* Character Arrays (Strings)
* Basic Program Structure in C

## Key Notes

* Strings are used to store Name, USN, and Branch.
* `scanf()` is used to accept user input.
* `printf()` is used to display formatted output.
* Proper formatting improves the readability of program output.
